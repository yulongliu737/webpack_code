# Project Summary

This repository contains a sample Webpack project demonstrating a typical front-end build setup.

## Structure
- **config/** – Development and production Webpack configuration files.
- **public/** – Static assets and the HTML entry point.
- **src/** – Application source code including JavaScript modules, style files, images, and fonts.

## Scripts
- `npm run dev` – Start the development server with hot reloading.
- `npm run build` – Create an optimized production build.

This summary provides a quick overview of the project's purpose and layout for new contributors.
